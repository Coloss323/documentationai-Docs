---
title: Cannot Upload License Document
description: Fix issues uploading pharmacy licenses.
---

## Symptoms

- Error message during upload
- File does not appear in the pharmacy profile

## Causes

- Unsupported file format
- File size exceeds 5MB

## Solution

<Steps>
  <Step title="Check file format">
    Only PDF and PNG formats supported.
  </Step>
  <Step title="Reduce file size">
    Compress file below 5MB before uploading.
  </Step>
  <Step title="Retry upload">
    Go to **Pharmacies > Edit > Documentation**, then upload again.
  </Step>
</Steps>
