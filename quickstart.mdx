## Prerequisites

Before starting, ensure you have:

<Callout kind="info">

- A valid email address for account registration
- Basic details about your pharmacy organization (name, address, license number)
- Admin access to your organization's systems for initial configuration

</Callout>

## Create and Activate Your Account

Follow these steps to set up your C3 Access account.

<Steps>
  <Step title="Visit the signup page" icon="external-link">
    Navigate to `https://dashboard.example.com/signup` in your browser.
  </Step>
  <Step title="Enter your details" icon="user-plus">
    Provide your email, organization name, and create a secure password. Agree to the terms of service.
  </Step>
  <Step title="Verify your email" icon="mail">
    Check your inbox for a verification email from C3 Access. Click the activation link to confirm your account.
  </Step>
</Steps>

<Callout kind="tip">
  Use a strong password with at least 12 characters, including uppercase, lowercase, numbers, and symbols.
</Callout>

## Sign In and Explore the Dashboard

Once activated, sign in to access your dashboard.

<Tabs>
  <Tab title="First Login" icon="log-in">
    Go to `https://dashboard.example.com/login`, enter your credentials, and complete any two-factor authentication if enabled.
  </Tab>
  <Tab title="Dashboard Overview" icon="layout">
    The dashboard shows key sections: Compliance Status, Tasks, Reports, and Settings. Use the sidebar to navigate.
  </Tab>
</Tabs>

## Set Up Your First Compliance Profile

Create a profile to track your pharmacy's compliance.

<Steps>
  <Step title="Navigate to Profiles" icon="file-plus">
    From the dashboard, click `Profiles` > `New Profile`.
  </Step>
  <Step title="Enter Organization Details" icon="edit-3">
    Input your pharmacy name, address, NPI number, and license details. Upload any required documents.
  </Step>
  <Step title="Configure Initial Checks" icon="check-circle">
    Select standard compliance checks like DEA registration and state licensing. Save and activate the profile.
  </Step>
</Steps>

## Initial Configuration via API (Optional)

For programmatic setup, use the C3 Access API to create profiles.

<CodeGroup tabs="JavaScript,Python,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/profiles', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'Main Pharmacy',
      address: '123 Pharmacy St, City, ST 12345',
      npi: '1234567890'
    })
  });
  const profile = await response.json();
  console.log(profile);
  ```
  ```python
  import requests

  headers = {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
  }
  data = {
      'name': 'Main Pharmacy',
      'address': '123 Pharmacy St, City, ST 12345',
      'npi': '1234567890'
  }
  response = requests.post('https://api.example.com/v1/profiles', json=data, headers=headers)
  profile = response.json()
  print(profile)
  ```
  ```bash
  curl -X POST https://api.example.com/v1/profiles \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "name": "Main Pharmacy",
      "address": "123 Pharmacy St, City, ST 12345",
      "npi": "1234567890"
    }'
  ```
</CodeGroup>

<ParamField path="profiles" param-type="POST" required="true">
  Endpoint to create compliance profiles.
</ParamField>

<ResponseField name="id" field-type="string" required="true">
  Unique profile identifier.
</ResponseField>

<ResponseField name="status" field-type="string">
  Current compliance status (`active`, `pending`).
</ResponseField>
