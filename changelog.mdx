## Recent Updates

Stay informed about the latest improvements to C3 Access, your compliance platform for pharmacy organizations. Review release notes below to understand new features, bug fixes, and any required actions.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

- Introduced real-time audit trail notifications via email and in-app alerts

## Improvements

- Enhanced dashboard performance with lazy loading for compliance reports
- Updated pharmacy license validation to include new state regulations

## Bug Fixes

- Fixed export issues with large CSV compliance datasets
- Resolved intermittent failures in multi-location report generation

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core compliance dashboard with controlled substance logs
- User role-based access for pharmacy managers and staff

## Breaking Changes

- Updated API endpoints from `/v0` to `/v1` paths
- Deprecated legacy report formats; migrate to new JSON exports

</Update>

## Migration Guide

If you encounter breaking changes from recent updates, follow these steps to update your integration seamlessly.

<Steps>
  <Step title="Update API Endpoints" icon="code">
    Replace all `/v0` paths with `/v1` equivalents.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      // Old
      const response = await fetch('https://api.example.com/v0/compliance');
      
      // New
      const response = await fetch('https://api.example.com/v1/compliance');
      ```
      ```python
      import requests
      
      # Old
      response = requests.get('https://api.example.com/v0/compliance')
      
      # New
      response = requests.get('https://api.example.com/v1/compliance')
      ```
    </CodeGroup>
  </Step>
  <Step title="Regenerate Tokens" icon="key">
    Invalidate old tokens and generate new ones from the dashboard.
  </Step>
  <Step title="Test Reports" icon="check-circle">
    Run sample compliance reports to verify data integrity.
  </Step>
</Steps>

<Callout kind="alert">
  Back up your data before migrating. Contact support if you need assistance with custom integrations.
</Callout>
